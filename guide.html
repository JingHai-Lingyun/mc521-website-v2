第一章：交友，是在服务器游玩的第一步

刚进服最重要的事情就是找个小伙伴

怎么找呢？


1. 在公屏直接刷屏找人(这种刷屏是允许的)

例如:“谁要一起玩 我tpa过去！”

[主动你们就有故事~]



2. 让服主帮你安排一个小伙伴,只要服主在线就可以喔

[高效率 免费安排]





第二章：发展方向

新人该从哪一步开始发展？


1. 如果你已经很久没玩MC了

那么前期的话可以先当做单机玩,了解高版本新知识与特性


2. 如果你认为你有基础知识

那么你就把菜单的每一个功能都仔细看一遍

主城的NPC功能也都看一遍

完成后你就搞懂该怎么去玩本服了




第三章：常用指令

圈地 - /res create 领地名

私聊 - /msg 玩家名 内容

玩家市场 - /ca help

玩家地标 - /pw help

拜师 - /twm help

结婚 - /marry help


还有更多指令，尽在 君庭阁工具箱 (左侧)




第四章：附魔也是一种新玩法

本服有200多种附魔书


以下是获取附魔书的渠道

1. 用原版附魔台附魔出来

2. 附魔书抽奖宝箱抽出来

3. 通过与玩家交易获得

4. 通过粉丝礼包 / 游戏活动等其他渠道获得




第五章：向上攀升 之 阶级系统

升级阶级，需要你花费时间和精力达到不同的标准

当然，升级后的回报也是十分丰厚的


具体有什么回报？

1. 解锁 / 扩容随身背包

2. 扩大圈地范围，降低圈地费用

除此之外，还有很多权限等你解锁哦~


升阶要求和奖励具体信息，尽在 君庭阁工具箱 (右侧)




第六章：个人天赋：属性提升好帮手

通过对应的行为，提升自己的天赋，成就更好的自己！


增幅生效在：

生命值 - 最高到40血 (20❤)

攻击力 - 最高到150%伤害 (+50%)

防御力 - 最高到125%防御 (+25%)


特别注意：技能面板中点击右上角，看到的才是各项属性加成




最后的总结

以上就是大概的指引

后面会随着玩法的推出慢慢更新
